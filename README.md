### Over ChristelijkePodcasts

ChristelijkePodcasts is een eenvoudige web-gebaseerde applicatie ontworpen om gebruikers een verzameling van christelijke podcasts te tonen. Of je nu nieuw bent in de wereld van christelijke podcasts of al langer geïnteresseerd bent, ChristelijkePodcasts biedt een gebruiksvriendelijke ervaring met toegang tot verschillende shows en onderwerpen. De site is te bezoeken via: [https://www.christelijkepodcasts.com/](https://www.christelijkepodcasts.com/)

### 🌟 Features

- **Podcast Ontdekking**: Ontdek een breed scala aan christelijke podcasts gesorteerd op verschillende categorieën en onderwerpen.
- **Favorieten**: Bewaar favoriete podcasts voor eenvoudig toegang en luisteren.
- **Podcast Details/Omleiding**: Bekijk gedetailleerde informatie over afleveringen, door op de podcast te klikken en omgeleid te worden naar de podcast op Spotify.
- **Trending Podcasts**: Toont populaire en trending christelijke podcasts op basis van gebruikersinteractie.
- **Filters en Zoeken**: Gebruik filters zoals categorieën, spreker en duur om je ideale podcasts te vinden.

### 💻 Tech Stack

ChristelijkePodcasts maakt gebruik van moderne technologieën voor een optimale prestatie en gebruikerservaring:

- **Next.js (TypeScript)**: Hoofdkader voor het bouwen van de app, ondersteund door server-side rendering en statische site generatie.
- **Vercel**: Hosting- en deployplatform geoptimaliseerd voor Next.js-applicaties.
- **TailwindCSS**: Utility-first CSS-framework voor snelle, consistente styling.
- **ShadCN**: UI-stylingbibliotheek voor modulaire en toegankelijke componenten, wat zorgt voor een samenhangend ontwerp.

### 📄 Pagina's

- **Home**: De hoofdpagina die een overzicht biedt van populaire en aanbevolen christelijke podcasts.
- **Categorieën**: Bekijk podcasts op basis van verschillende onderwerpen zoals onderwijs, aanbidding en bijbelstudie.
- **Trending**: Bekijk de meest populaire podcasts van dit moment.
- **Help/FAQ**: Toegang tot veelgestelde vragen en ondersteuningsbronnen.

### 🌐 API

ChristelijkePodcasts gebruikt API's om gegevens van podcasts en gerelateerde inhoud te fetchen, wat zorgt voor een up-to-date en gevarieerd aanbod. De belangrijkste API die ChristelijkePodcasts gebruikt is de [Spotify WebAPI](https://developer.spotify.com/documentation/web-api). Met behulp van deze API worden alle podcasts gefetched waarna vervolgens weergegeven op het startscherm en andere pagina's. Daarnaast gebruikt ChristelijkePodcasts ook de [Bijbel API](https://rapidapi.com/ajith/api/holy-bible) om de bijbeltekst op het startscherm te fetchen en weer te geven.

### 🌟 Features

- **Podcast Discovery**: Discover a wide range of Christian podcasts sorted by various categories and topics.
- **Favorites**: Save your favorite podcasts for easy access and listening.
- **Podcast Details/Redirect**: View detailed information about episodes by clicking on the podcast, and be redirected to the podcast on Spotify.
- **Trending Podcasts**: Displays popular and trending Christian podcasts based on user interaction.
- **Filters and Search**: Use filters such as categories, speaker, and duration to find your ideal podcasts.

### 💻 Tech Stack

ChristelijkePodcasts uses modern technologies to ensure optimal performance and a great user experience:

- **Next.js (TypeScript)**: Main framework for building the app, supported by server-side rendering and static site generation.
- **Vercel**: Hosting and deployment platform optimized for Next.js apps.
- **TailwindCSS**: Utility-first CSS framework for fast, consistent styling.
- **ShadCN**: UI styling library for modular and accessible components, ensuring a cohesive design.

### 📄 Pages

- **Home**: The main page, showcasing popular and recommended Christian podcasts.
- **Categories**: Browse podcasts based on various topics such as education, worship, and Bible study.
- **Trending**: View the most popular podcasts at the moment.
- **Help/FAQ**: Access frequently asked questions and support resources.

### 🌐 API

ChristelijkePodcasts uses APIs to fetch podcast data and related content, ensuring up-to-date and diverse offerings. The main API used by ChristelijkePodcasts is the [Spotify WebAPI](https://developer.spotify.com/documentation/web-api). This API fetches all podcasts, which are then displayed on the homepage and other pages. Additionally, ChristelijkePodcasts also uses the [Bible API](https://rapidapi.com/ajith/api/holy-bible) to fetch and display Bible verses on the homepage.
